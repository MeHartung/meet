Meet App

The Meet App leverages serverless functions to ensure high responsiveness, which is crucial for providing real-time updates. By utilizing serverless technology, the app can automatically scale to handle high loads without compromising performance. Additionally, serverless functions enable seamless integration of geolocation services and authentication via the Google Calendar API, allowing the app to provide location-specific event details.

Feature 1: Show/Hide Event Details

Users have the ability to reveal or hide detailed information about events with a simple button click. This provides control over how much information is visible at any given time.

    •	Scenario: You are viewing the event details page.
    •	Action: You click the button to toggle the visibility of event details.
    •	Result: The event’s detailed information is either expanded or collapsed, depending on its previous state.

Feature 2: Specify Number of Events

The app allows users to control how many events are displayed at once, giving you flexibility in how much content you see on the events listing page.

    •	Scenario: You are on the events listing page.
    •	Action: You select a specific number of events to display using a drop-down menu.
    •	Result: The page refreshes and shows only the number of events you specified.

Feature 3: Use the App When Offline

The app is designed to be functional even without an internet connection. If you lose connection, you will still have access to cached data and previously viewed events.

    •	Scenario: You’ve accessed the app while connected to the internet.
    •	Action: You lose your internet connection while using the app.
    •	Result: The app notifies you that it is operating in offline mode and continues to display cached data and events that were viewed before going offline.

Feature 4: Add an App Shortcut to the Home Screen

For easier access, users can add the Meet App directly to their home screen. This makes launching the app more convenient, especially for frequent users.

    •	Scenario: You’ve installed the app on your mobile device.
    •	Action: You choose the option to add the app as a shortcut to your home screen.
    •	Result: The app icon appears on your home screen, allowing for quicker access.

Feature 5: Display Charts Visualizing Event Details

To help users better understand event data, the app includes a feature to display event details in a chart format. This visual representation makes it easier to analyze event information.

    •	Scenario: You are viewing the events list page.
    •	Action: You tap the “View Charts” button.
    •	Result: A chart is displayed, offering a visual overview of the event details.
